# Forge 2 Kanban Board Application

This repository contains the Forge 2 Qualifier application build, featuring:
- **Backend:** Laravel REST API utilizing SQLite.
- **Frontend:** React + Vite interactive board.

Built autonomously using Hermes and OpenClaw.

## Running Locally

### Backend
1. Go to `/backend`
2. Copy `.env.example` to `.env`
3. Run `composer install`
4. Run `php artisan migrate`
5. Run `php artisan serve`

### Frontend
1. Go to `/frontend`
2. Run `npm install`
3. Run `npm run dev`
